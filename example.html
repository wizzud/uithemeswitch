Left Switcher

This is the default uithemeswitcher with no options set. So themes are unsorted, showing large icons, etc, etc.

Middle Switcher

This uithemeswitcher shows the Remember item, only collapses on mouseout of the theme list, always shows the current theme name, and its themes are unsorted.
  • closeMouseout: 'list';
  • showRemember : 1;
  • sorted : false;
  • themeRollover : 0;

Right Switcher

The 3rd uithemeswitcher shows both the Remember and the Reset items, and is set to list the smaller theme icons to save some space.
  • compact : true;
  • showRemember : 1;
  • showReset : 1;
